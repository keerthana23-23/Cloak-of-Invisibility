This is a Python project that demonstrates an invisibility cloak effect using OpenCV.
By wearing a colored cloth (red, blue, or pink), the object becomes invisible on the live webcam feed.

Features
Real-time cloak effect using OpenCV and HSV color masking.
Supports multiple cloak colors: Red, Sky Blue, Pink.
Background is captured dynamically for realistic invisibility.
Video recording functionality included (.avi format).
